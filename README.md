
# 🤖 AI Support Ticket Classification System

# 📌 Project Overview

This project is a Machine Learning + NLP based web application that automatically classifies customer support tickets into different categories such as:

* Billing Inquiry
* Technical Issue
* Account Access
* Delivery Issue
* Refund Request

The system helps businesses reduce manual work, improve customer support efficiency, and prioritize issues faster.

# 🚀 Features

✅ Text preprocessing using NLP
✅ Stopword removal and lemmatization
✅ TF-IDF feature extraction
✅ Machine Learning ticket classification
✅ Live support ticket prediction
✅ Interactive Streamlit dashboard
✅ Confusion matrix and performance visualization
✅ Priority distribution analysis

# 🧠 Machine Learning Concepts Used

* Natural Language Processing (NLP)
* Text Cleaning
* TF-IDF Vectorization
* Logistic Regression
* Text Classification
* Model Evaluation
* Confusion Matrix
  
# 📂 Dataset Used

Customer Support Ticket Dataset containing:

* Ticket Subject
* Ticket Description
* Ticket Type
* Ticket Priority

# 📊 Model Workflow

1. Load dataset
2. Clean and preprocess text
3. Convert text into numerical vectors using TF-IDF
4. Train Logistic Regression model
5. Predict ticket categories
6. Display results on Streamlit dashboard
